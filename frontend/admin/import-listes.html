Aide - Import par Listes
Qu'est-ce qu'un export par listes ?

Dans MyLudo, vous pouvez creer des listes pour organiser vos jeux (Ex: "Expert", "Famille", "Puzzles", "Petite Enfance").

L'export par listes contient une colonne liste indiquant a quelle liste appartient chaque jeu.

Gestion des doublons

Un meme jeu peut apparaitre plusieurs fois s'il est dans plusieurs listes. Liberteko :

  • Detecte les doublons (par EAN ou titre)
  • Cree le jeu une seule fois
  • Ajoute toutes ses listes comme thematiques
Listes → Thematiques

Chaque nom de liste (Ex: "Expert") devient une thematique dans Liberteko. Les thematiques permettent :

  • La recherche intelligente
  • Le filtrage du catalogue
  • Les recommandations
Prerequis : Votre fichier doit avoir une colonne liste en premiere position.
