Aide - Import MyLudo
Format MyLudo

L'export MyLudo utilise le separateur point-virgule (;). Allez dans Options > Exporter pour obtenir votre fichier CSV.

Colonnes reconnues automatiquement
  • Titre (obligatoire)
  • Sous-titre
  • Type (base/extension)
  • EAN
  • Edition (annee)
  • Editeur(s)
  • Auteur(s)
  • Illustrateur(s)
  • Joueur(s)
  • Duree
  • Age(s)
  • Categorie(s)
  • Theme(s)
  • Mecanisme(s)
  • Langues
  • Univers / Gamme
Gestion de collection
  • Date d'acquisition
  • Prix indicatif / Prix d'achat
  • Emplacement
  • Etat (neuf, bon, etc.)
  • Proprietaire
  • Dimensions / Poids
Options de collection
  • Gratuit / Cadeau
  • Prive / Protege
  • Organise / Personnalise
  • Figurines peintes
  • Commentaire
  • Reference / Referent
Astuce : Les jeux existants sont detectes par leur code EAN ou leur titre. Choisissez "Mettre a jour" pour synchroniser votre collection.
